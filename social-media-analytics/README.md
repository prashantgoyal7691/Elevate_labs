# 📱 Social Media Analytics Backend

## 📌 Overview
This project implements a SQL-based backend system to track social media engagement including posts, likes, and comments.

## 🎯 Objectives
- Design relational schema
- Track engagement metrics
- Use triggers for automation
- Generate analytics using SQL

## 🛠 Tools
- PostgreSQL
- SQL

## 📂 Structure
```
social-media-analytics/
│
├── sql/
│   ├── 01_schema.sql
│   ├── 02_data.sql
│   ├── 03_triggers.sql
│   ├── 04_views.sql
│   └── 05_analysis.sql
│
├── outputs/
│   ├── screenshots/
│   └── results.txt
│
└── README.md
```

## 🔍 Features
- Like tracking system
- Engagement scoring
- Ranking using window functions
- Automated like count updates using triggers

## 📊 Insights
- Identifies top posts
- Measures engagement
- Tracks user activity
